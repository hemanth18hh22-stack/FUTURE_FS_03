# FUTURE_FS_03 – Gym Website (Local Business Website)

## Project Overview

This project is a **Gym Website** developed as part of the **Future Interns Full Stack Web Development Internship – Task 3**.

The goal of this project is to create a **professional website for a local business**. In this case, a **fitness gym**. The website helps promote the gym, provide information about services, and attract potential customers.

---

## Features

* Modern homepage design
* About the gym section
* Membership plans
* Trainers information
* Contact section
* Responsive layout for better user experience

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## Project Structure

```
gym-website
│
├── index.html
├── style.css
└── script.js
```

---

## How to Run the Project

1. Download or clone the repository.

2. Open the project folder.

3. Run the website by opening:

```
index.html
```

in your web browser.

---

## Business Value

This website helps a local gym business by:

* Increasing online visibility
* Providing information about membership plans
* Allowing customers to easily contact the gym
* Promoting fitness services
